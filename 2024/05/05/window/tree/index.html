tree

发表于2024-05-05|更新于2024-09-05|window

window

tree [drive:][path] [/F][/a]

参数

  • drive 盘符
  • path 文件路径
  • /F 递归列出所有文件
  • /A 只查看文件夹, 忽略文件

tree /F > tree.txt

生成的文件目录树形结构写入到 tree.txt

tree /A > tree.txt

查看文件夹,忽略文件

路径

tree D:/a > D:/b/tree.txt 把 D 盘下 a 的目录机构接到 D 盘下 b 文件的 tree.txt 文件中 如果指定的目录中已经存在该文件则覆盖
